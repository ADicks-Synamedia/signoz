# Docs

Team-owned documentation. Each subdirectory has a single owning agent.

| Directory | Owner | Purpose |
|---|---|---|
| `decisions/` | Chronicler (storage), BossArchitect (content) | ADRs |
| `chronicler/` | Chronicler | Per-session notes |
| `reviews/` | Reviewer | Post-implementation audits per Change |
| `gaps/` | CompleterFinisher | Phase-boundary gap reports |
| `mediator/` | Mediator | Conflict rulings |
| `phases/` | PragmaticTPM | Per-Phase detail files |

Top-level files you will see as the project matures:

| File | Owner |
|---|---|
| `architecture.md` | BossArchitect |
| `roadmap.md` | PragmaticTPM |

These are not created by the bootstrap; they are created by the owning agent in their first session.
